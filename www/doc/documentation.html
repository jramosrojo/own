Aoko HTML Template

Documentation by Andy Lu

Thanks for buying "Aoko", the premium website template. In this help documentation, I will walk you through the template files, this information may help you with customizing the template for your own use.

HTML Structure

This template is based on foundation framework, to understand what the "row", "column", and "container" classes mean, please reference to the foundation documentation.

CSS Structure

The css files are mainly based(with some modifications) on foundation framework. 

          <!--
  CSS files:

    1. app.css
    2. font-awesome.css
    3. fontello.css
    4. foundation.css
    5. linea-arrows.css
    6. linea-styles.css
    7. responsive.css
    8. slick-theme.css
    9. slick.css

  The 'app.css' is the main style file for everything, the following is the structure of it.

    1. Base
    2. Logo
    3. Home and nav
    4. Content
    5. Client
    6. Icon-text

    9. Work

    12. Case study

Javascript Structure

The file "app.js" is written in a way required by the TerrificJS framework which enables you to modularize the jQuery code by solely relying on the OOCSS naming conventions. 

                <!--
  Javascript files:
    1. slick.min.js
    2. jquery.appear.js
    3. jquery.sequence-min.js
    4. app.js

  The 'app.js' is the main style file for everything, the following is the structure of it.

    (function($){

      // initialize things ...

    })(jQuery);

-->

PHP Script

There is a php script "send.php" that will process the form submission and send emails, you need to change the email recipient to make it work. 

          <!--
    PHP scripts:
    1. send.php

    send.php
    replace YOUR-EMAIL-ADDRESS with your email address

    $to = "YOUR-EMAIL-ADDRESS";

    $name = $_POST['name'];
    $subject = "You've got a message";

    ...

-->

Menu SVG Customization

The SVG shape should not be larger than 600px by 337px. You could make a shape you want in Adobe Illustrator, fill the shape with black and select no stroke as shown below.

Then save the file as SVG, click save.
